# SecureShield – Advanced Malware Detection System

SecureShield is a professional, futuristic, and secure malware detection dashboard built with React, Node.js, and Express.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (Running locally or via Atlas)

### 🛠️ Installation

1. **Backend Setup**
   ```bash
   cd server
   npm install
   # Create a .env file with MONGODB_URI and PORT
   npm start
   ```

2. **Frontend Setup**
   ```bash
   cd client
   npm install
   npm run dev
   ```

## ✨ Key Features
- **Full System Scan Module**: Interactive scanning with real-time progress.
- **Threat Detection Dashboard**: Detailed analysis of malware entry points and impact.
- **Attack Vector Analysis**: Visual diagrams showing the path of infection.
- **System Safety Status**: Real-time health score and engine status.
- **Detailed Logs**: Chronological history of all security events.
- **Malware Removal**: Quarantine and removal modules with instant feedback.

## 🎨 Tech Stack
- **Frontend**: React.js, TypeScript, Framer Motion, Recharts, Lucide Icons.
- **Backend**: Node.js, Express, MongoDB.
- **Styling**: Premium Vanilla CSS with Glassmorphism and Neon accents.

## 🛡️ Security
This system architecture uses best practices for RESTful APIs and modern frontend security patterns.
